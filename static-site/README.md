# Static Website Version

This folder is a standalone static version of the Punch For Purpose website.

## Open locally

1. Open `static-site/index.html` directly in your browser.

## Important differences from Next.js version

- No Node.js required.
- No API routes.
- Forms and Stripe are placeholders in static mode.
- Event pages are static HTML files.

The original full Next.js build is unchanged in `src/`.
